# 数据模型与接口需求

## 1. 数据模型设计原则

1. MVP 优先保证核心链路清晰。
2. 业务数据和模型调用日志分开存储。
3. 文档原文、文档切片、向量索引需要可追溯。
4. 所有 AI 生成内容需要记录来源和生成时间。
5. API Key 等敏感信息必须加密存储。

## 3. API 接口需求

接口路径仅为建议，可按实际框架调整。

## 3.1 知识库接口

### 创建知识库

POST /api/knowledge-bases

请求：

```json
{
  "name": "大模型应用开发",
  "description": "学习 RAG、Agent、模型 API 等内容"
}
```

响应：

```json
{
  "id": "kb_001",
  "name": "大模型应用开发",
  "description": "学习 RAG、Agent、模型 API 等内容",
  "document_count": 0
}
```

### 获取知识库列表

GET /api/knowledge-bases

### 获取知识库详情

GET /api/knowledge-bases/{id}

### 更新知识库

PUT /api/knowledge-bases/{id}

### 删除知识库

DELETE /api/knowledge-bases/{id}

## 3.2 文档接口

### 上传文档

POST /api/knowledge-bases/{id}/documents

请求：

- multipart/form-data
- files: 文件数组

响应：

```json
{
  "documents": [
    {
      "id": "doc_001",
      "file_name": "rag-intro.pdf",
      "status": "uploaded"
    }
  ]
}
```

### 获取文档列表

GET /api/knowledge-bases/{id}/documents

### 获取文档详情

GET /api/documents/{id}

### 删除文档

DELETE /api/documents/{id}

### 重新解析文档

POST /api/documents/{id}/reparse

## 3.3 问答接口

### 创建问答

POST /api/chat

请求：

```json
{
  "knowledge_base_id": "kb_001",
  "session_id": "session_001",
  "question": "RAG 的核心流程是什么？",
  "model_provider": "deepseek",
  "model_name": "deepseek-chat"
}
```

响应：

```json
{
  "session_id": "session_001",
  "answer": "RAG 的核心流程包括文档解析、切分、向量化、检索和生成回答。",
  "citations": [
    {
      "document_id": "doc_001",
      "document_name": "rag-intro.pdf",
      "chunk_id": "chunk_001",
      "snippet": "RAG combines retrieval with generation..."
    }
  ]
}
```

### 获取会话列表

GET /api/chat/sessions

### 获取会话消息

GET /api/chat/sessions/{id}/messages

## 3.4 任务接口

### 创建任务

POST /api/tasks

### 获取任务列表

GET /api/tasks

支持查询参数：

- status
- priority
- knowledge_base_id
- source_type

### 更新任务

PUT /api/tasks/{id}

### 完成任务

POST /api/tasks/{id}/complete

### 删除任务

DELETE /api/tasks/{id}

### 从回答生成任务

POST /api/tasks/generate

请求：

```json
{
  "source_message_id": "msg_001",
  "content": "你接下来应该学习 FastAPI、完成 API 调用 Demo，并尝试搭建向量库。"
}
```

响应：

```json
{
  "tasks": [
    {
      "title": "学习 FastAPI 基础",
      "description": "掌握路由、请求参数、响应模型和接口调试。",
      "priority": "medium"
    }
  ]
}
```
