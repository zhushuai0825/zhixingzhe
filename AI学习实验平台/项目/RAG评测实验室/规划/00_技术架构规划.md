# 技术架构规划

## 1. 当前架构

```text
纯前端 HTML + CSS + JavaScript
```

使用本地教学数据模拟 RAG 评测。

## 2. 后续真实架构

```text
RAG 系统日志
  -> 评测数据集
  -> 指标计算服务
  -> 人工标注或 LLM Judge
  -> 评测报告
```

## 3. 后续可扩展指标

- NDCG。
- HitRate。
- Answer Relevance。
- Context Precision。
- Context Recall。
- Faithfulness。
- Hallucination Rate。
- Latency。
- Cost。

## 4. 后续工程重点

- 保存每次实验配置。
- 对比不同版本 RAG 的指标。
- 记录失败样本。
- 生成优化建议。
- 支持导出评测报告。
