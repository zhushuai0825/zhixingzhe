# Agent 工程实验室

这个项目用于学习 Agent 开发：目标如何变成计划，计划如何调用工具，工具结果如何被观察、评估、反思，最后如何生成用户真正需要的结果。

## 你会学到什么

- Agent 和普通聊天机器人的区别。
- Tool Calling 是什么。
- Agent 状态、工具、控制流、停止条件是什么。
- 为什么 Agent 需要证据门和反思检查。
- Agent 怎么结合 RAG、Graph RAG、Rerank。
- 后续如何升级到 LangGraph。

## 当前实现

当前是纯前端教学版，不调用真实大模型。

你可以：

- 输入目标。
- 切换 Agent 类型。
- 开关 RAG 工具、Graph RAG 工具、反思检查。
- 调整最大工具调用步数。
- 调整证据阈值。
- 新增知识片段。
- 新增工具。
- 查看完整工具调用轨迹。
- 点击每一步查看输入、输出和解释。
- 在执行轨迹图里查看 Agent 状态流。
