# Agent 工程实验室技术架构规划

## 1. 当前架构

```text
index.html
  -> 页面结构
styles.css
  -> 布局、响应式、控件样式
app.js
  -> Agent 模拟运行、工具检索、轨迹渲染、canvas 可视化
```

## 2. 当前存储

当前使用浏览器 `localStorage` 保存新增知识和工具。

## 3. 后续真实工程

下一阶段可以升级为：

```text
FastAPI
  -> Agent Run API
  -> Tool API
  -> Run Log API

PostgreSQL
  -> agent_runs
  -> agent_steps
  -> agent_tools
  -> agent_knowledge

LangGraph
  -> planner
  -> tool nodes
  -> evaluator
  -> reflection
  -> final
```

## 4. 和其他模块关系

- RAG 模块：作为 `rag_search` 工具。
- Graph RAG 模块：作为 `graph_search` 工具。
- Rerank 模块：作为 `rerank_evidence` 工具。
- RAG 评测模块：评估 Agent 最终回答和工具调用质量。
