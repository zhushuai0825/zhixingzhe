# Agent 工程实验室测试验收清单

## 1. 页面检查

- 页面能从平台左上角切换进入。
- 项目总览、Agent 实验台、执行轨迹图、学习路线、面试题都能打开。
- 手机宽度下不出现明显横向溢出。

## 2. Agent 实验台

- 修改目标后，运行轨迹和最终结果会变化。
- 切换 Agent 类型后，计划会变化。
- 关闭 RAG 工具后，不再出现 RAG 检索步骤。
- 关闭 Graph RAG 工具后，不再出现 Graph RAG 步骤。
- 关闭反思检查后，不再出现 Reflect 步骤。
- 修改证据阈值后，证据门判断可能变化。
- 新增知识片段后，检索结果可以使用新知识。
- 新增工具后，工具箱数量会增加。

## 3. 执行轨迹

- 每个轨迹卡片可以点击。
- 当前步骤详情会展示输入和输出。
- 执行轨迹图不空白。
- 鼠标悬停轨迹节点能看到解释。

## 4. 文档检查

- `README.md` 说明项目定位和打开方式。
- `01_Agent零基础学习手册.md` 能解释 Agent 和 Tool Calling 的区别。
- `02_工程实现说明.md` 能说明数据结构和后续 LangGraph 升级方式。
- `03_测试验收清单.md` 能指导后续修改验收。
