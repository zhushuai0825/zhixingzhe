# 技术架构规划

## 1. 当前版本

当前版本是纯前端教学版：

```text
HTML + CSS + JavaScript
```

不依赖后端，不调用真实 Rerank 模型。

## 2. 后续真实版本

后续可以升级为：

```text
前端实验台
  -> FastAPI 后端
  -> 向量库召回
  -> Rerank 模型
  -> 证据门
  -> 大模型回答
```

## 3. 可接入模型

- BGE Reranker。
- Cohere Rerank。
- Jina Reranker。
- 本地 cross-encoder。
- 大模型打分。

## 4. 关键工程问题

- Top K 召回多少合适。
- Rerank 后保留多少证据。
- 分数阈值如何设。
- 证据不足时如何拒答。
- Rerank 延迟如何控制。
- Rerank 结果如何评测。
