# 测试验收清单

## 1. 打开检查

启动平台：

```bash
cd /Users/zhushuai/Downloads/学习/AI学习实验平台
python3 -m http.server 8099
```

打开：

```text
http://127.0.0.1:8099
```

切换到 `Rerank 重排序实验室`。

必须看到：

- 项目总览。
- Rerank 实验台。
- 排序可视化。
- 学习路线。
- 面试题。

## 2. 交互检查

在实验台里修改问题：

```text
Rerank 为什么会增加成本？
```

检查：

- 向量库初筛结果会刷新。
- Rerank 后证据会刷新。
- 详情区会显示成本相关解释。
- 排序可视化会重新绘制。

## 3. 参数检查

修改：

```text
向量召回 Top K = 4
最终保留 Top N = 2
```

检查：

- 初筛列表显示 4 条。
- Rerank 后证据显示 2 条。
- Prompt 证据门只展示最终保留证据。

## 4. 策略检查

切换策略：

- 平衡。
- 严格。
- 语义。

检查：

- 严格策略更看重可回答性。
- 语义策略更接近向量初筛顺序。
- 排序变化能在画布上看出来。

## 5. 项目切换检查

点击左上角项目名称，确认可以切到：

- 文本切分实验室。
- Embedding 与相似度。
- 向量库实验室。
- 知行者 AI 实验室。
- 推荐系统相关实验室。

```bash
node --check /Users/zhushuai/Downloads/学习/AI学习实验平台/项目/Rerank重排序实验室/前端原型/app.js
```

- Canvas 绘制异常。
