# 推荐系统核心技术实验室

这个模块用于学习推荐系统的四条基础路线：

1. 标签/内容推荐：根据物品标签和内容相似度推荐。
2. ItemCF：根据物品共现关系推荐。
3. UserCF：根据相似用户喜欢过什么推荐。
4. SVD：用矩阵分解学习用户和物品的隐向量。

## 当前能力

- 使用可编辑的本地教学数据，不需要后端。
- 可切换目标用户。
- 可新增用户、物品、喜欢行为，也可以删除喜欢行为。
- 可对比标签推荐、ItemCF、UserCF、SVD。
- 展示 Top N 推荐结果、推荐分和推荐理由。
- 用画布展示当前用户历史行为如何连接到候选推荐。
- 提供推荐系统面试题。

## 数据怎么和其他推荐实验室联动

本模块和 `标签推荐实验室`、`UserCF实验室`、`SVD矩阵分解实验室` 共用一份浏览器本地数据。

你在推荐系统总览里新增一个物品，切到标签推荐、UserCF、SVD 也能继续用；反过来也一样。这样设计是为了让你用同一批数据对比不同算法：

- 标签推荐会看物品标签是否命中用户画像。
- ItemCF 会看物品之间有没有共同喜欢用户。
- UserCF 会看哪些用户和当前用户行为相似。
- SVD 会看用户隐向量和物品隐向量的点积分。

## 和 ItemCF 专项实验室的关系

`基于物品的协同过滤ItemCF` 是专项模块，重点深入 ItemCF、行为共现、推荐分贡献和推荐可视化。

本模块是总览模块，重点对比多个推荐算法，让你先知道每种算法适合什么场景。
