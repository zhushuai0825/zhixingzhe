## 2. 当前系统定位

知行者 AI 实验室不是普通聊天网页，而是一个学习型 AI 工程项目。它重点让你看懂三件事：

- 文档怎么进入知识库。
- 问题怎么命中知识库。
- Agent 怎么围绕目标调用 RAG 工具并生成最终结果。

## 3. 当前已完成能力

- 前端原型工作台。
- RAG 实验室。
- 可视化实验台。
- 数据中心。
- Agent 实验室。
- PostgreSQL 存储知识库、文档、切片、RAG 记录、Agent 记录。
- Chroma 存储向量和向量索引。
- 本地学习版 Embedding。
- BM25 关键词检索。
- Hybrid RAG 检索。
- 大模型基于引用生成最终回答。
- Agent 运行轨迹、最终回答、引用来源展开、任务生成。
- 知识库和文档删除。

## 6. 重要原则

以后继续做这个项目时，保持下面的方向：

- 先把一个能力跑通，再追求复杂。
- 每个 AI 能力都要有可视化过程。
- RAG 必须展示引用来源。
- Agent 必须展示最终结果，也必须保留运行轨迹。
- 文档、代码、页面要同步更新。
