# 系统使用手册

这份文档告诉你如何从 0 开始使用知行者 AI 实验室。

## 1. 打开系统

先确认服务已经启动：

```bash
cd /Users/zhushuai/Downloads/知行者AI实验室
docker compose up -d
```

启动后端：

```bash
cd /Users/zhushuai/Downloads/知行者AI实验室/后端
source .venv/bin/activate
uvicorn app.main:app --host 127.0.0.1 --port 8010
```

启动前端：

```bash
cd /Users/zhushuai/Downloads/知行者AI实验室/前端原型
python3 -m http.server 8088
```

浏览器打开：

```text
http://127.0.0.1:8088
```

## 2. 推荐使用顺序

第一次使用按这个顺序走：

1. 进入 `RAG 实验室`。
2. 创建知识库。
3. 上传或粘贴一份文档。
4. 预览切片。
5. 导入知识库。
6. 去 `数据中心` 看文档和切片是否入库。
7. 去 `可视化实验台` 输入问题，看命中哪些向量点。
8. 去 `Agent 实验室` 输入目标，看大模型最终回答、引用来源和运行轨迹。

## 3. RAG 实验室怎么用

RAG 实验室主要解决一个问题：

```text
文档如何变成 AI 可以检索和引用的知识库？
```

你在这里会看到：

- 原始文档。
- 文本切片。
- 切片长度和重叠长度。
- 向量预览。
- 入库流程。
- 检索实验参数。

推荐先用 `.md` 或 `.txt` 文件学习，因为纯文本最容易观察。PDF 也可以上传，但扫描版 PDF 暂时读不到文字，需要以后接 OCR。

## 4. 可视化实验台怎么用

可视化实验台主要解决一个问题：

```text
我问一句话以后，系统到底命中了哪些文档片段？
```

你在这里可以：

- 输入问题。
- 选择检索模式。
- 看向量空间 3D 展示。
- 看问题向量和文档向量的距离。
- 看命中的 Top K 片段。
- 看每条命中的分数。

当前支持三种检索模式：

- `vector`：只看向量相似度。
- `keyword`：只看 BM25 关键词匹配。
- `hybrid`：向量 + 关键词融合，当前主推模式。

## 5. 数据中心怎么用

数据中心主要用于检查真实入库数据：

- 知识库列表。
- 文档列表。
- chunk 切片列表。
- 真实向量预览。
- 删除知识库。
- 删除文档。

如果你上传后发现结果不对，先来数据中心看：

- 文档是否存在。
- 切片数量是否正常。
- 切片内容是否被截断得太奇怪。
- 向量预览是否出现。

## 6. Agent 实验室怎么用

Agent 实验室不是普通问答页。它模拟的是：

```text
用户目标
  -> Agent 制定计划
  -> 调用 RAG 检索工具
  -> 读取证据
  -> 判断证据是否足够
  -> 大模型生成最终回答
  -> 生成下一步任务
  -> 反思最小行动
```

运行后先看 `Agent 最终结果`。

这里是你真正需要看的答案，包括：

- 大模型最终回答。
- 使用的模型。
- 引用来源。
- 下一步建议。
- 生成任务。

再看 `工具调用轨迹`。

这里用于学习 Agent 怎么运行：

- Plan：计划。
- Retrieve：检索。
- Observe：读取证据。
- Evaluate：评估证据。
- Act：生成回答和任务。
- Reflect：反思下一步。

最后看 `步骤调试详情`。

这里不是最终答案，而是每个节点的输入输出日志。

## 7. 引用来源怎么看

Agent 最终结果里的引用来源可以展开。

每条引用包含：

- 文件名。
- chunk 序号。
- 相关度分数。
- 命中原文。
- 来源方式，例如 `vector + keyword`。
- chunk_id。

如果你怀疑大模型回答不可信，就展开引用来源，看它是否真的来自你的知识库。

## 8. 什么时候说明系统正常

下面这些都正常，说明系统闭环跑通：

- 上传文档后，数据中心能看到文档。
- 文档有多个 chunk。
- chunk 有向量预览。
- 可视化实验台能命中 Top K。
- Agent 最终结果能看到大模型回答。
- 引用来源能展开看到原文。
- 后端 `/health` 返回 PostgreSQL 和 Chroma 都正常。

## 9. 当前边界

当前版本还不是企业生产系统，主要用于学习和演示。

已知边界：

- Embedding 是本地学习版哈希向量，不是真正语义模型。
- PDF 只支持可复制文字的 PDF。
- 暂不支持图片 OCR。
- 暂不支持 Word。
- 暂不支持多用户权限。
- 暂不支持正式部署认证。
