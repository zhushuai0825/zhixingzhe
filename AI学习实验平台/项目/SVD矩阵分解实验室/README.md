# SVD 矩阵分解实验室

SVD / 矩阵分解是推荐系统里非常重要的一类思想：把用户和物品都放进同一个低维隐向量空间，然后用点积预测用户对物品的喜欢程度。

核心流程：

```text
用户-物品行为矩阵
  -> 学习用户隐向量
  -> 学习物品隐向量
  -> 用户向量 · 物品向量
  -> 预测喜欢程度
  -> Top N 推荐
  -> 3D 隐向量空间可视化
```

## 学什么

- 用户-物品矩阵是什么。
- 为什么推荐矩阵通常很稀疏。
- 隐向量是什么。
- 点积为什么能表示匹配程度。
- SVD 和协同过滤的区别。
- 矩阵分解为什么是深度推荐的基础思想之一。

## 怎么自己测试

页面里的 `实验数据中心` 可以新增物品和喜欢行为。新增物品时，系统会根据标签生成一个教学版 2 维物品向量。

你可以这样试：

1. 新增 `Graph RAG 实战课`，标签填 `AI, RAG, 图谱`。
2. 新增 `推荐系统召回课`，标签填 `推荐, 算法, 召回`。
3. 给当前用户添加不同喜欢行为。
4. 观察用户隐向量怎么变化。
5. 看候选物品的点积分为什么会变高或变低。

这里不是完整训练版 SVD，而是教学版隐向量实验台。它的目标是让你先看懂“用户向量、物品向量、点积预测”这条主线。
