## 1. 在平台中的定位

ItemCF 负责推荐算法入门主线：

- 用户行为矩阵。
- 物品共现。
- 物品相似度。
- 推荐分计算。
- Top N 推荐。
- 推荐解释和可视化。

## 2. 平台入口

```text

```

```text

```

即可打开本项目。

## 3. 运行方式

ItemCF 前端默认可以使用浏览器本地算法，不启动后端也能学习。

如果要使用 API 模式：

```bash
cd /Users/zhushuai/Downloads/学习/AI学习实验平台/项目/基于物品的协同过滤ItemCF/后端
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --host 127.0.0.1 --port 8020
```

然后在页面点击：

```text
连接后端
```
