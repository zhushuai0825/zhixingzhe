# 测试验收清单

每次修改后按这份清单检查。

## 1. 数据库检查

启动：

```bash
cd /Users/zhushuai/Downloads/学习/AI学习实验平台/项目/基于物品的协同过滤ItemCF
docker compose up -d
```

检查：

```bash
docker compose ps
docker exec itemcf-postgres psql -U itemcf -d itemcf_lab -c "select (select count(*) from users) users, (select count(*) from items) items, (select count(*) from interactions) interactions;"
```

期望：

```text
users = 6
items = 31
interactions = 49
```

## 2. 后端检查

启动：

```bash
cd /Users/zhushuai/Downloads/学习/AI学习实验平台/项目/基于物品的协同过滤ItemCF/后端
source .venv/bin/activate
uvicorn app.main:app --host 127.0.0.1 --port 8020
```

检查：

```bash
curl http://127.0.0.1:8020/health
```

期望：

```json
{
  "status": "ok",
  "postgres": true,
  "users": 6,
  "items": 31,
  "interactions": 49
}
```

检查状态接口：

```bash
curl "http://127.0.0.1:8020/api/state?user_id=u2"
```

必须包含：

- `dataSource: postgres`
- `users`
- `items`
- `interactions`
- `recommendations`
- `itemSimilarityPoints`

## 3. 前端检查

打开统一平台：

```bash
cd /Users/zhushuai/Downloads/学习/AI学习实验平台
python3 -m http.server 8099
```

浏览器打开：

```text
http://127.0.0.1:8099
```

切换到 `ItemCF 学习实验室`。

检查：

- 状态显示 `PostgreSQL · 49 条真实行为`。
- 推荐可视化台能显示目标用户、匹配画布和推荐结果。
- 切换推荐可视化台的用户后，推荐结果和画布同步变化。
- 推荐可视化台可以新增当前用户喜欢的物品。
- 推荐可视化台可以创建自定义物品，并自动加入当前用户喜欢。
- 推荐可视化台可以删除当前用户已喜欢的物品。
- 新增或删除后，推荐结果、矩阵、画布同步刷新。
- 点击推荐项后，匹配画布高亮对应历史物品到候选物品的路径。
- 鼠标悬停匹配画布节点时，可以看到完整标题和说明。
- 用户下拉框可以切换用户。
- Top N 推荐能显示。
- 候选物品得分表能显示。
- 推荐分贡献明细能显示。
- 用户-物品矩阵能显示。
- 相似度矩阵能显示。
- 共现矩阵能显示。
- 3D 物品相似度空间能显示。
- 3D 视图可以拖拽旋转。
- 3D 视图可以滚轮缩放。
- 点击推荐项后，3D 视图高亮推荐物品和来源物品。

## 4. 写入检查

在页面给当前用户新增一个喜欢行为。

检查：

- 页面推荐结果变化。
- 状态仍显示 PostgreSQL。
- 数据库 interactions 数量增加。

命令：

```bash
docker exec itemcf-postgres psql -U itemcf -d itemcf_lab -c "select count(*) from interactions;"
```

检查：

- items 回到 31。
- interactions 回到 49。
- U2 推荐结果恢复。
