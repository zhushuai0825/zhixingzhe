# Graph RAG 图谱实验室

这个项目用于学习 Graph RAG：为什么要把知识从“文本片段”升级成“实体 + 关系 + 原文证据”。

## 你会学到什么

- 普通向量 RAG 和 Graph RAG 的区别。
- 实体、关系、路径、图遍历分别是什么。
- 为什么图谱检索适合法律、金融风控、医疗知识关联等多跳问题。
- 为什么图谱边不能直接当答案，最终仍然要回到原文 chunk。
- Graph RAG 如何和向量检索、Hybrid 检索、Rerank 一起工作。

## 当前实现

当前是纯前端教学版，不依赖后端和真实图数据库。

你可以：

- 输入问题。
- 调整最大跳数。
- 切换起点命中策略。
- 新增实体。
- 新增关系。
- 新增证据片段。
- 查看多跳路径。
- 查看回答草稿和引用证据。
- 在 3D 图谱里拖拽旋转、滚轮缩放、悬停查看实体信息。

## 学习定位

建议放在这条路线里：

```text
文本切分 -> Embedding -> 向量库 -> Rerank -> RAG 评测 -> Graph RAG -> 知行者完整系统
```

原因是：Graph RAG 不是 RAG 入门第一步，它是解决“普通检索已经不够准、不够会关联”之后的进阶能力。
