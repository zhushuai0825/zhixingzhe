# Graph RAG 图谱实验室产品需求

## 1. 背景

用户已经学习了文本切分、Embedding、向量库、Rerank 和 RAG 评测。下一步需要理解复杂知识关联场景：为什么有些问题不能只靠向量相似度解决。

## 2. 产品目标

做一个可交互的 Graph RAG 学习实验室，让用户直观看到：

- 文档知识如何变成实体。
- 实体之间如何形成关系。
- 问题如何命中起点实体。
- 系统如何沿关系做多跳扩散。
- 路径如何回到原文证据。
- 证据如何支撑回答。

## 3. 核心功能

- 项目总览。
- 图谱实验台。
- 3D 图谱空间。
- 学习路线。
- 面试题。
- 新增实体。
- 新增关系。
- 新增证据片段。
- 重置演示数据。
- 路径和证据可视化。

## 4. 成功标准

- 输入变了，路径和证据会变化。
- 用户能自己添加数据并观察结果。
- 3D 图谱能拖拽、缩放、悬停、点击。
- 文档能解释 Graph RAG 的工程流程和学习路线。
