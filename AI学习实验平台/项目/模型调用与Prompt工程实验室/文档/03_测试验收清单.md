# 模型调用与 Prompt 工程实验室测试验收清单

## 1. 页面检查

- 页面能从平台左上角切换进入。
- 项目总览、调用实验台、调用链路图、学习路线、面试题都能打开。

## 2. 调用实验台

- 修改 system prompt 后，诊断结果会变化。
- 修改上下文后，token 和成本会变化。
- 关闭 JSON 输出后，模型输出变成自然语言。
- 开启引用校验但清空上下文时，会出现校验失败或重试。
- 关闭自动重试后，失败不会自动修复。
- 切换模型策略后，成本和延迟会变化。
- 调整 max output 后，输出 token 会变化。

## 3. 可视化

- 调用链路图不空白。
- 链路节点能展示 Prepare、Request、Generate、Validate、Retry 等步骤。
- 悬停节点能看到解释。

## 4. 文档检查

- README 说明项目定位和打开方式。
- 学习手册能解释 Prompt、结构化输出、重试和成本。
- 工程说明能解释后续真实 API 设计。
