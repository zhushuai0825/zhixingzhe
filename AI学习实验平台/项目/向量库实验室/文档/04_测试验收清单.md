# 测试验收清单

## 1. 页面加载

- 打开平台入口。
- 从左上角项目名称切换到向量库实验室。
- 页面标题、左侧导航、项目总览正常显示。

## 2. 后端连接

启动后端：

```bash
cd /Users/zhushuai/Downloads/学习/AI学习实验平台/项目/向量库实验室/后端
source .venv/bin/activate
uvicorn app.main:app --host 127.0.0.1 --port 8050
```

检查：

```bash
curl http://127.0.0.1:8050/health
```

期望：

- 返回 `status: ok`。
- 页面顶部显示 `FastAPI · 8050 在线`。

## 3. Upsert

操作：

- 输入新 ID：`chunk-test`。
- 输入文本：`向量库可以保存文本向量并按相似度查询。`
- 点击 `Upsert 入库`。

期望：

- collection 列表出现 `chunk-test`。
- 选中记录向量出现非零维度。

## 4. Query

操作：

- 查询：`向量数据库怎么保存 embedding？`
- Top K 选择 3。
- 点击 `查询 Top K`。

期望：

- 命中结果出现。
- 分数最高的结果和向量库、embedding 有关。
- “为什么命中”显示共享语义维度。

## 5. Metadata 过滤

操作：

- source 过滤输入 `vector_db.md`。
- 再查询。

期望：

- 只返回 source 为 `vector_db.md` 的记录。

## 6. Delete

操作：

- 删除 `chunk-test`。

期望：

- collection 列表不再显示 `chunk-test`。
- 再查询不会命中该记录。

## 7. 3D 可视化

操作：

- 切到“命中可视化”。
- 鼠标拖拽画布。
- 鼠标滚轮缩放。
- 悬停圆点。

期望：

- 画布旋转、缩放正常。
- 悬停能看到记录文本、来源和分数。

## 8. 离线兜底

操作：

- 停止 8050 后端。
- 刷新页面。

期望：

- 页面不白屏。
- 顶部显示 `离线教学向量库`。
- 基础 upsert/query/delete 仍可在浏览器本地演示。
