# Embedding 学习资料索引

## 1. 先学概念

必须理解：

- 文本不能直接计算相似度。
- 文本要先变成数字向量。
- 向量是很多维数字组成的列表。
- 相似度计算比较的是两个向量，不是原始文本。

## 2. 再学公式

重点学：

```text
余弦相似度
点积
欧氏距离
L2 归一化
```

## 3. 再学工程

重点看：

- embedding 接口输入输出。
- 向量维度。
- 批量 embedding。
- 向量缓存。
- embedding 成本。
- embedding 模型升级后的重建索引。
