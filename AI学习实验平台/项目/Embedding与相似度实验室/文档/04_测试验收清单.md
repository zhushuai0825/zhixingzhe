# 测试验收清单

## 1. 后端检查

```bash
curl http://127.0.0.1:8030/health
```

期望：

```json
{
  "status": "ok",
  "dim": 16
}
```

## 2. 接口检查

```bash
curl -X POST http://127.0.0.1:8030/api/compare \
  -H 'Content-Type: application/json' \
  -d '{"query":"embedding 是什么","metric":"cosine","topK":2,"candidates":[{"id":"d1","text":"embedding 把文本变成向量"},{"id":"d2","text":"Docker 用于部署服务"}]}'
```

期望：

- 返回 query。
- 返回 results。
- results 有 score。
- d1 通常应排在 d2 前面。

## 3. 前端检查

打开：

```text
http://127.0.0.1:8099/平台前端/
```

切换到：

```text
Embedding 与相似度实验室
```

检查：

- 页面能打开。
- 左上角项目切换正常。
- 查询文本能修改。
- 候选文本能修改。
- 点击运行后 Top K 更新。
- token 标签显示。
- 向量条形图显示。
- 相似度画布显示。
- 鼠标放到画布节点能看完整文本。

## 4. JS 语法检查

```bash
node --check 前端原型/app.js
node --check ../../平台前端/app.js
```

```bash
python3 -m py_compile 后端/app/main.py
```
