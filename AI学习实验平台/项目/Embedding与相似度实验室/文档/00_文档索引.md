## 2. 当前项目回答什么问题

本项目回答：

- 文本为什么要变成向量？
- token 是什么？
- embedding 向量是什么？
- 向量为什么要归一化？
- 余弦相似度、点积、欧氏距离有什么区别？
- RAG 检索为什么本质上是向量相似度排序？
- 查询文本到底查的是哪里？
- 3D 相似度空间应该怎么看？

## 3. 当前项目边界

当前只做：

```text
文本 -> token -> embedding -> 相似度 -> Top K
```

不做：

- 文档上传。
- 文本切分。
- 向量数据库。
- Rerank。
- 大模型回答。

这些会在后续独立项目继续做。
